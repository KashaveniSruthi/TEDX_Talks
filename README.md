# TEDx Talks Curated Web Application 

A Python-based web application that displays a curated collection of TEDx Talks with embedded YouTube videos.  
The project dynamically generates an interactive webpage where users can browse talks and watch them directly within the application.

---

##  Project Overview

This project demonstrates the use of **Python Object-Oriented Programming (OOP)** and **web technologies** to build a simple content-based application.

The application stores information about TEDx talks such as title, summary, thumbnail image, video URL, and publish date. Using Python, the project dynamically generates an HTML webpage that displays all curated talks in a visually appealing layout.

Users can click on any talk thumbnail to instantly watch the TEDx video without leaving the webpage.

## ▶ How to Run the Project

### Step 1: Clone the repository


git clone https://github.com/KashaveniSruthi/TEDX_Talks.git


### Step 2: Navigate to project folder


cd tedx-talks-project


### Step 3: Run the Python file

### Step 4: View the webpage

##  Output

Each talk tile displays:

- Talk title
- Description
- Publish date
- Talk thumbnail
- Embedded YouTube video player

Clicking on a talk thumbnail opens the video directly within the webpage.
